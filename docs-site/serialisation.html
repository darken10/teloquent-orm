Sérialisation

Contrôlez ce qui apparaît dans toJSON() / toObject() : masquez les champs sensibles, exposez des attributs calculés, ou limitez à une liste blanche.

toJSON / toObject

const user = await User.find(1);
user.toJSON();    // applique casts, hidden/visible, appends
JSON.stringify(user); // utilise toJSON automatiquement

hidden : masquer des champs

class User extends Model {
  static hidden = ["password", "remember_token"];
  declare password: string;
}

(await User.find(1)).toJSON();
// { id, name, email, ... } — sans password ni remember_token
Indispensable pour une API : sans hidden, toJSON() exposerait toutes les colonnes, y compris les mots de passe.

visible : liste blanche

Si visible est non vide, seuls ces attributs sont sérialisés.

class Token extends Model {
  static visible = ["id", "value"];
}
// toJSON() ne renverra que { id, value }

appends : attributs calculés

Ajoute des accessors (sans colonne en base) à la sortie JSON.

class User extends Model {
  static appends = ["display_name"];
  getDisplayNameAttribute() {
    return `★ ${this.getAttribute("name")}`;
  }
}

(await User.find(1)).toJSON();
// { id, name, ..., display_name: "★ Inoussa ZERBO" }

Surcharges par instance

user.makeVisible("password").toJSON(); // force l'affichage (cette instance)
user.makeHidden("email").toJSON();     // masque en plus (cette instance)
user.append("score").toJSON();         // ajoute un accessor (cette instance)

Ces méthodes renvoient l'instance (chaînables) et n'affectent pas les autres.

Relations & collections

Les relations chargées sont sérialisées automatiquement, et le filtrage hidden/visible s'y applique aussi (une relation peut être masquée par son nom).

const users = await User.with("posts").get();
users.toJSON();
// [ { id, name, posts: [ { id, title, ... } ] }, ... ]

class Account extends Model {
  static hidden = ["secrets"]; // masque la relation "secrets"
}

Récapitulatif

OutilEffet
static hiddenRetire des attributs de la sortie
static visibleListe blanche stricte (si non vide)
static appendsAjoute des attributs calculés (accessors)
makeHidden / makeVisible / appendSurcharges par instance
