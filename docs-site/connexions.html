Connexions

Le ConnectionManager est le registre central des connexions. On enregistre une ou plusieurs connexions au démarrage, puis les modèles les utilisent automatiquement.

Enregistrer une connexion

import { ConnectionManager } from "teloquent-orm";

// SQLite (fichier ou ":memory:")
await ConnectionManager.addConnection({ driver: "sqlite", database: "app.sqlite" });

PostgreSQL & MySQL

// PostgreSQL
await ConnectionManager.addConnection({
  driver: "pgsql",
  host: "127.0.0.1", port: 5432,
  username: "user", password: "secret", database: "app",
});

// MySQL / MariaDB
await ConnectionManager.addConnection({
  driver: "mysql",
  host: "127.0.0.1", port: 3306,
  username: "root", password: "secret", database: "app",
});

Connexions multiples

On peut enregistrer plusieurs connexions nommées et choisir laquelle utiliser.

await ConnectionManager.addConnection({ driver: "sqlite", database: "app.sqlite" });          // "default"
await ConnectionManager.addConnection({ driver: "pgsql", /* ... */ }, "reporting");

ConnectionManager.connection();            // connexion par défaut
ConnectionManager.connection("reporting"); // connexion nommée
ConnectionManager.setDefault("reporting"); // change la connexion par défaut

Pour qu'un modèle utilise une connexion précise :

class Report extends Model {
  static connectionName = "reporting";
}

Transactions

transaction() exécute un callback : commit si tout réussit, rollback automatique si une exception est levée. Les transactions imbriquées sont gérées (un seul commit/rollback au niveau le plus externe).

await ConnectionManager.connection().transaction(async () => {
  const user = await User.create({ name: "A" });
  await Post.create({ title: "Bienvenue", user_id: user.id });
  // si une erreur survient ici, les deux insertions sont annulées
});

Requêtes brutes

const conn = ConnectionManager.connection();

// SELECT
const rows = await conn.select("select * from users where id = ?", [1]);

// Écriture (INSERT/UPDATE/DELETE/DDL)
const result = await conn.statement("update users set name = ? where id = ?", ["X", 1]);
result.affectedRows; // nombre de lignes affectées

Fermer les connexions

await ConnectionManager.closeAll();

Connexion personnalisée

Pour un driver maison (ou des tests), on peut enregistrer une Connection déjà construite.

import { Connection, ConnectionManager, PostgresGrammar } from "teloquent-orm";

const conn = new Connection(monDriverCustom, new PostgresGrammar());
ConnectionManager.register(conn, "default");
