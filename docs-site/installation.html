Installation

Installez Teloquent et le driver de votre base de données, puis configurez TypeScript.

Prérequis

  • Node.js ≥ 18 (le repli SQLite natif node:sqlite nécessite Node ≥ 22).
  • Un projet ESM ("type": "module" dans votre package.json).
  • TypeScript ≥ 5 (recommandé, mais Teloquent fonctionne aussi en JS).

Installer le paquet

npm install teloquent

Installer un driver

Teloquent ne dépend d'aucun driver par défaut : choisissez celui de votre SGBD.

SGBDPaquetCommande
SQLitebetter-sqlite3npm install better-sqlite3
PostgreSQLpgnpm install pg
MySQL / MariaDBmysql2npm install mysql2
Astuce SQLite : si better-sqlite3 ne se compile pas, Teloquent se replie automatiquement sur le module natif node:sqlite (Node ≥ 22, aucune compilation).

Configurer TypeScript

Deux options sont indispensables au bon fonctionnement de l'accès magique aux attributs.

tsconfig.json 
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "Bundler",
    "strict": true,
    "experimentalDecorators": true,
    "useDefineForClassFields": false
  }
}
Important : useDefineForClassFields: false empêche les champs déclarés (declare name: string) d'être définis comme propriétés réelles, ce qui permet au Proxy d'intercepter les accès aux attributs. Sans cela, user.name renverra undefined.

Premier exemple

Créez un fichier et lancez-le avec tsx (npm i -D tsx) ou après compilation.

app.ts 
import { Model, ConnectionManager, Schema } from "teloquent-orm";

await ConnectionManager.addConnection({ driver: "sqlite", database: ":memory:" });

await Schema().create("users", (t) => {
  t.increments("id");
  t.string("name");
  t.timestamps();
});

class User extends Model {
  declare id: number;
  declare name: string;
}

const u = await User.create({ name: "Inoussa ZERBO" });
console.log(u.id, u.name);            // 1 Inoussa ZERBO
console.log((await User.all()).length); // 1
npx tsx app.ts

Vérifier l'installation

Si l'exemple affiche 1 Inoussa ZERBO, tout est prêt. Passez à la page Connexions pour configurer votre base réelle.
