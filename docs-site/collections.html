Collections

Les requêtes de modèles renvoient une Collection : un tableau JavaScript enrichi de méthodes pratiques. Toutes les méthodes natives d'Array restent disponibles.

Obtenir une collection

const users = await User.all();              // Collection<User>
const actifs = await User.where("active", true).get();

users.length;          // comme un tableau
users.map((u) => u.name);
for (const u of users) { /* ... */ }

Méthodes utilitaires

MéthodeDescription
first()Premier élément ou null
last()Dernier élément ou null
isEmpty()true si vide
pluck(key)Extrait une colonne en tableau
keyBy(key)Map indexée par une clé
groupBy(key)Map de groupes
toArray()Tableau simple
toJSON()Sérialise chaque élément

Exemples

const users = await User.all();

users.first();                 // User | null
users.isEmpty();               // false

users.pluck("email");          // ["a@b.com", "c@d.com", ...]

const byId = users.keyBy("id"); // Map<number, User>
byId.get(1);                    // User #1

const byRole = users.groupBy("role"); // Map<string, User[]>
byRole.get("admin")?.length;

users.toJSON();                // applique hidden/visible/appends sur chaque modèle

Chaînage avec Array

const noms = (await User.where("active", true).get())
  .filter((u) => u.getAttribute("age") >= 18)
  .map((u) => u.name)
  .sort();

Construire une collection

import { Collection } from "teloquent";

const c = Collection.fromArray([{ id: 1 }, { id: 2 }]);
c.pluck("id"); // [1, 2]
