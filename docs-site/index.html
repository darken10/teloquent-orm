Teloquent

Un ORM TypeScript Active Record inspiré de Laravel Eloquent, avec un typage fort et le support de SQLite, MySQL et PostgreSQL.

TypeScript ESM · Node ≥ 18 SQLite MySQL PostgreSQL Licence MIT

Démarrage rapide

import { Model, ConnectionManager } from "teloquent-orm";

await ConnectionManager.addConnection({ driver: "sqlite", database: "app.sqlite" });

class User extends Model {
  declare id: number;
  declare name: string;
  posts() { return this.hasMany(Post); }
}

const user = await User.create({ name: "Inoussa ZERBO" });
const actifs = await User.where("is_active", true).with("posts").get();

Pourquoi Teloquent

  • API quasi identique à Eloquent : find, where, create, save, hasMany, with, Schema…
  • Typage fort : génériques et autocomplétion — l'avantage sur PHP.
  • Multi-SGBD : une couche Grammar par dialecte ; valeurs toujours liées (anti-injection).
  • Complet : relations (dont N-N), eager loading imbriqué, soft deletes, scopes, pagination, migrations.
