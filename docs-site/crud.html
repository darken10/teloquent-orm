CRUD

Créer, lire, mettre à jour et supprimer des enregistrements, façon Eloquent.

Créer

// create() : instancie + sauvegarde, renvoie le modèle avec son id
const u = await User.create({ name: "Inoussa ZERBO", email: "z@africasys.com" });
u.id; // 1

// new + save()
const u2 = new User({ name: "Awa" });
await u2.save();
L'id auto-incrémenté est récupéré automatiquement après l'insertion, quel que soit le SGBD (lastInsertId pour SQLite/MySQL, RETURNING pour PostgreSQL).

Lire

await User.find(1);            // User | null
await User.findOrFail(1);      // User (throw si absent)
await User.all();              // Collection<User>

await User.where("is_active", true).get();      // Collection<User>
await User.where("email", "z@a.com").first();   // User | null
await User.query().where("age", ">", 18).firstOrFail();

Mettre à jour

// Sur une instance (dirty tracking : seules les colonnes modifiées sont écrites)
const u = await User.findOrFail(1);
u.name = "Inoussa Z.";
await u.save();

// Raccourci fill + save
await u.update({ name: "X", email: "x@a.com" });

// En masse via le query builder
await User.query().where("is_active", false).update({ status: "archived" });

Supprimer

const u = await User.findOrFail(1);
await u.delete();   // true

// En masse
await User.query().where("spam", true).delete();
Si le modèle active les soft deletes, delete() positionne deleted_at au lieu de supprimer la ligne.

Rafraîchir & recharger

await u.refresh(); // recharge les attributs depuis la base

Helpers pratiques

Voir la page Helpers CRUD pour firstOrCreate, updateOrCreate, upsert, increment/decrement.

await User.firstOrCreate({ email: "z@a.com" }, { name: "Inoussa ZERBO" });
await User.updateOrCreate({ email: "z@a.com" }, { name: "Inoussa Z." });
await user.increment("votes", 5);
